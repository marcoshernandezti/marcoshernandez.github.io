## 🚀 Proyectos Destacados

### 🏙 Sistema de Gestión de Comunas  
🔗 [Repositorio](https://github.com/marcoshernandezti/ProyectoComunas)  
Aplicación desarrollada en .NET 8 con arquitectura en tres capas:
